Git

  • git init - инициализация нового локального git-репозитория.
  • git clone - получение копии удалённого репозитория.
  • git commit - фиксация текущих изменений с сообщением.
  • git push - отправка изменений на удалённый сервер.
  • mgit pull - загрузка обновлений из удалённого сервера и слияние с локальной версией.
  • git branch - управление ветками проекта.
